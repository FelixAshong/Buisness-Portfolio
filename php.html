Why Php

PHP is a server side scripting language. that is used to develop Static websites or Dynamic websites or Web applications. PHP scripts are faster than other technology because it reduces the workload of the server. Php provides a large number of databases with error handling and correcting technique and this will help the Php to be the one of the most popular languge for web development.

PHP is cross platform; this means you can deploy your application on a number of different operating systems such as windows, Linux, Mac OS etc. PHP is open source and free and faster than ASP or JSP.Php is regular updated to keep abreast with the latest technology trends. Most web hosting servers support PHP by default unlike other languages such as ASP that need IIS. This makes PHP a cost effective choice. Other benefit that you get with PHP is that it’s a server side scripting language; this means you only need to install it on the server and client computers requesting for resources from the server do not need to have PHP installed; only a web browser would be enough. PHP has in built support for working hand in hand with MySQL; this doesn’t mean you can’t use PHP with other database management systems. You can still use PHP with Postgres Oracle MS SQL Server ODBC etc.
